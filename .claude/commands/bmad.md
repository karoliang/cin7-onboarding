# BMAD Commands

This command provides access to BMAD (Build More, Architect Dreams) functionality.

Usage:
- `/bmad build` - Build agent XML files from YAML sources
- `/bmad status` - Show installation status
- `/bmad list` - List available modules
- `/bmad install` - Install BMAD Core agents and tools
- `/bmad update` - Update existing BMAD installation
- `/bmad uninstall` - Remove BMAD installation

Examples:
- `/bmad build example-agent` - Build specific agent
- `/bmad status --verbose` - Show detailed installation status
- `/bmad list --all` - List all available modules including details
