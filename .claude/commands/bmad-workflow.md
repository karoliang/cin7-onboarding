# BMAD Workflow Management

This command helps manage BMAD workflows in the project.

Usage:
- `/bmad-workflow create <workflow-name>` - Create a new workflow template
- `/bmad-workflow run <workflow-name>` - Execute a workflow
- `/bmad-workflow list` - List all workflows in the project
- `/bmad-workflow status <workflow-name>` - Check workflow status

Examples:
- `/bmad-workflow create onboarding-pipeline` - Create a new onboarding workflow
- `/bmad-workflow run data-pipeline` - Execute the data pipeline workflow
- `/bmad-workflow list` - Show all available workflows
