# BMAD Agent Management

This command helps manage BMAD agents in the project.

Usage:
- `/bmad-agent create <agent-name>` - Create a new agent template
- `/bmad-agent build <agent-name>` - Build a specific agent
- `/bmad-agent list` - List all agents in the project
- `/bmad-agent validate <agent-name>` - Validate agent configuration

Examples:
- `/bmad-agent create data-processor` - Create a new data processor agent
- `/bmad-agent build example-agent` - Build the example agent
- `/bmad-agent list` - Show all available agents
